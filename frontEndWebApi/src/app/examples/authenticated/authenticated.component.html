Authenticated component

This component is only accessible for authenticated users

Logging out will cause user to be redirected to home screen if visiting route accessible only to authenticated users

To guard a route you can use RoleGuard, specify the allowed roles using the data property (roles will be retrieved from API token)

      {
        path: 'editorauthenticated',
        component: EditorAuthenticatedComponent,
        canActivate: [RoleGuard],
        data: {
          roles: ['editor', 'admin']
        }
      }

To guard a route you can also use ClaimGuard, specify the required claims using the data property (claims will be retrieved from API token)

      {
        path: 'authenticated',
        component: AuthenticatedComponent,
        canActivate: [ClaimGuard],
        data: {
          claims: { roles: ['editor', 'admin']}
        }
      }
